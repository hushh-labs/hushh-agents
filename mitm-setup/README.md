## 🚀 STEP-BY-STEP SETUP

---

### STEP 1: MacBook pe Proxy Start Karo

Terminal mein run karo:

```bash
cd /Users/ankit.singh/Desktop/PBL/hushh-agents/mitm-setup
./start-mitm.sh
```

Ya manually:

```bash
mitmweb --listen-port 8080 --web-port 8081 --set console_eventlog_verbosity=info
```

**Kya hoga:**
- Port `8080` pe proxy server chalega (iPhone isse connect karega)
- Port `8081` pe web dashboard khulega browser mein (`http://localhost:8081`)
- Terminal pe live traffic dikhega

> ⚠️ **IMPORTANT**: Proxy pehle start karo, phir iPhone configure karo. Nahi toh `mitm.it` nahi khulega.

---

### STEP 2: Confirm — MacBook ka IP Address

Terminal mein check karo:

```bash
ipconfig getifaddr en0
```

**Current IP: `192.168.1.6`**

> ⚠️ Ye IP change ho sakta hai agar WiFi reconnect karo. Har baar verify karo.

---

### STEP 3: iPhone ko Proxy se Connect Karo

iPhone uthao aur ye steps follow karo:

#### 3.1 — Same WiFi pe Connect Karo
- iPhone aur MacBook **SAME WiFi network** pe hone chahiye
- Settings → Wi-Fi → Check karo ki same network hai

#### 3.2 — Proxy Configure Karo
1. **Settings** app kholo
2. **Wi-Fi** pe tap karo
3. Apne **connected WiFi network** ke aage **(i)** icon tap karo
4. Scroll down karo → **"Configure Proxy"** ya **"HTTP Proxy"** section dikhega
5. **"Manual"** select karo
6. Ye values daalo:

| Field | Value |
|-------|-------|
| **Server** | `192.168.1.6` |
| **Port** | `8080` |
| **Authentication** | OFF (toggle off rakho) |

7. **Save** tap karo (top-right corner)

**✅ Ab iPhone ka SAARA internet traffic MacBook se hoke jaayega!**

#### 3.3 — Quick Test
- iPhone pe Safari kholo
- Koi bhi website kholo (jaise `http://example.com`)
- MacBook pe mitmweb dashboard (`http://localhost:8081`) mein traffic dikhna chahiye
- Agar dikh raha hai → **Proxy connection SUCCESSFUL** ✅

---

### STEP 4: CA Certificate Download Karo (iPhone pe)

> ⚠️ Ye step zaroori hai HTTPS traffic dekhne ke liye. Without this, sirf HTTP requests dikhegi.

#### 4.1 — Certificate Download
1. iPhone pe **Safari** kholo (Chrome/Firefox kaam nahi karega!)
2. Address bar mein type karo: **`http://mitm.it`**
3. Ek page khulega with different OS options
4. **"Apple" (iOS)** button pe tap karo
5. Ek `.pem` file download hogi
6. Popup aayega: **"This website is trying to download a configuration profile. Do you want to allow this?"**
7. **"Allow"** tap karo

#### 4.2 — Certificate Install
1. **Settings** app kholo
2. Top pe ek banner dikhega: **"Profile Downloaded"** → Tap karo
   - Agar banner nahi dikha: **Settings → General → VPN & Device Management** jaao
3. **"mitmproxy"** profile dikhega → tap karo
4. **"Install"** tap karo (top-right)
5. iPhone ka **Passcode** daalo
6. Warning aayegi — **"Install"** phir se tap karo
7. **"Done"** tap karo

#### 4.3 — Certificate TRUST Enable Karo (⚠️ MOST IMPORTANT STEP!)

> Bahut log ye step bhool jaate hain! Certificate install ho gaya lekin trust nahi kiya toh HTTPS kaam nahi karega.

1. **Settings** app kholo
2. **General** → **About** → scroll down → **"Certificate Trust Settings"**
3. **"mitmproxy"** ke aage toggle **ON** karo
4. Warning popup aayega: **"Continue"** tap karo

**✅ Ab HTTPS traffic bhi decrypt hoga!**

---

### STEP 5: Verify — Sab Kaam Kar Raha Hai

## 🔧 TROUBLESHOOTING

### ❌ `mitm.it` nahi khul raha
- **Check**: Proxy correctly set hai? IP aur port sahi hai?
- **Check**: MacBook pe mitmproxy/mitmweb chal raha hai?
- **Check**: Dono devices same WiFi pe hain?
- **Fix**: Safari use karo (Chrome mein kabhi kabhi issue aata hai)

### ❌ HTTPS sites "Not Secure" / "Connection Not Private" error
- **Check**: Certificate install kiya? (Settings → General → VPN & Device Management)
- **Check**: Certificate TRUST enable kiya? (Settings → General → About → Certificate Trust Settings)
- **Fix**: Step 4.3 dobara follow karo

### ❌ Internet kaam nahi kar raha (proxy set karne ke baad)
- **Check**: MacBook pe mitmproxy chal raha hai? Agar band hai toh internet nahi chalega
- **Fix**: Proxy off karo (Settings → WiFi → (i) → Proxy → Off) aur mitmproxy phir start karo

### ❌ App traffic nahi dikh raha
- **Check**: App certificate pinning use kar raha hai? (hushh-agents web app mein nahi hai, toh ye issue nahi hona chahiye)
- **Check**: App sahi URL pe hit kar raha hai?

## 🧹 CLEANUP (Jab Kaam Ho Jaaye)

### iPhone se Proxy Hataao
1. Settings → Wi-Fi → (i) → Configure Proxy → **"Off"**

### Certificate Remove Karo
1. Settings → General → VPN & Device Management → mitmproxy → **"Remove Profile"**

### Certificate Trust Disable Karo
1. Settings → General → About → Certificate Trust Settings → mitmproxy → **Toggle OFF**

> ⚠️ **IMPORTANT**: Jab MITM session khatam ho jaaye, toh ZAROOR cleanup karo. Nahi toh:
> - Certificate installed rehne se security risk hai
> - Proxy on rehne se internet band ho jaayega (jab mitmproxy band hoga)
